## ğŸ”§ Troubleshooting

### "Speech recognition not supported"
**Solution:** Use Google Chrome or Microsoft Edge browser

### Microphone not working
**Solutions:**
1. Check browser permissions (allow microphone)
2. Make sure no other app is using your microphone
3. Try refreshing the page
4. Check your device's microphone settings

### "Connection error"
**Solutions:**
1. Check your internet connection
2. Wait 30-60 seconds and try again (server might be waking up)
3. Refresh the page
4. Try a different browser

### Friend can't join my room
**Check:**
1. Both using the EXACT same room code
2. Both have internet connection
3. Both granted microphone permission
4. Try generating a new room code

### Translation is wrong or weird
**Tips:**
1. Speak more clearly
2. Speak slower
3. Use simpler sentences
4. Avoid slang or idioms
5. Check if you selected the correct languages

### No sound when receiving messages
**Solutions:**
1. Check your device volume
2. Make sure sound is not muted
3. Check browser sound permissions
4. Try using headphones

### App is very slow
**Reasons:**
1. First use (server waking up) - wait 30-60 seconds
2. Poor internet connection
3. Too many people using the app
4. Try refreshing the page
